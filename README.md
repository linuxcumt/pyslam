# pyslam
Basic Sparse-Cholesky Graph SLAM solver implemented in python

Requirements
------------
* numpy >= 1.12
* scipy >= 0.18
* cython >= 0.25
* scikit-sparse >= 0.4.2
* suitesparse-dev

On Ubuntu 16.04:
```bash
sudo apt-get install libsuitesparse-dev
sudo -H pip install numpy scipy cython scikit-sparse
```
